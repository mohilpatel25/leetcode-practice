## 747. Largest Number At Least Twice of Others
Link: https://leetcode.com/problems/largest-number-at-least-twice-of-others/

```java
class Solution {
    public int dominantIndex(int[] nums) {
        int n = nums.length, max = 0;
        for(int i=1;i<n;i++)
            if(nums[i] > nums[max])
                max = i;
        for(int i=0;i<n;i++){
            if(nums[i] != nums[max]){
                if(2*nums[i] > nums[max])
                    return -1;
            }
        }
        return max;
    }
}
```
