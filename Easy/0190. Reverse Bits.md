## 190. Reverse Bits
Link: https://leetcode.com/problems/reverse-bits/

```java
public class Solution {
    // you need treat n as an unsigned value
    public int reverseBits(int n) {
        int rev = 0;
        for(int i=0;i<32;i++){
            rev = rev << 1;
            rev += n&1;
            n >>= 1;
        }
        return rev;
    }
}

```
