## 1688. Count of Matches in Tournament
Link: https://leetcode.com/problems/count-of-matches-in-tournament/

```java
class Solution {
    public int numberOfMatches(int n) {
        int matches = 0;
        while(n > 1){
            if(n%2 == 0){
                matches += n/2;
                n /= 2;
            }
            else{
                matches += (n-1)/2;
                n = n/2 + 1;
            }
        }
        return matches;
    }
}
```
